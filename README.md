# Catalogue for IMDB movies.

## Init database
`mongoimport --db imdb-catalogue --collection movies --file movies.json`

## Install deps/ Launch@8080
- Install server dependencies.

`npm install`

- Install client dependencies.

`cd client`

`npm install`

- Start server

`npm start`

- Start client

`cd client`

`npm start`
